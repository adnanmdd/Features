<h2>Smart Cooling System using XOR in Machine Learning</h2>

<h3>Overview</h3>

<p>
This project explains how the <strong>XOR (Exclusive OR)</strong> problem
is used in a <strong>smart cooling system</strong> with machine learning.
In a 2D feature space, XOR data cannot be separated using a single
straight line, making it a <strong>non-linear classification</strong> problem.
</p>

<p>
The system uses two inputs:
<strong>Temperature</strong> and <strong>Window Open</strong>.
The cooling system turns <strong>ON</strong> only when one condition is high and the other is low.
</p>

<h3>Outcome of the Project</h3>

<p>
The <strong>smart cooling system</strong> intelligently decides when to turn the cooling mechanism <strong>ON</strong> or <strong>OFF<strong> based on the conditions. This improves <strong>automation</strong>, <strong>energy efficiency</strong>, and
decision-making accuracy.
</p>
