  <h2>Feature Representation in Machine Learning</h2>

  <p>
    In machine learning, feature representation means how we describe data using numerical values (features) so that a model can learn patterns from it. 
    For example, in a 2D plane, two classes may overlap in such a way that a straight line cannot separate them.
    However, by transforming the data into a higher-dimensional space (such as 3D), the same data may become separable using a linear plane.
  </p>
